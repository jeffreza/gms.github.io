VIRCLIA® LOTUS

This fully automated random access and continuous loading CLIA analyser allows to run more than 95 infectious markers in a monotest format with total flexibility and no need of reagents on board. VirClia® Lotus brings a revolution to the infectious serology labs as it enables to perform in just one CLIA system low-volume parameters daily, alongside the laboratory routine.

Multiparametric immunoassay system for medium throughput

Fully automated random access and continuous loading CLIA analyser.
Allows to run more than 90 infectious markers in a monotest format with total flexibility and no need of reagents on board.
The first results are available within 70 minutes and then, every 35 seconds thereafter.
